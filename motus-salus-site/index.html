Project One

Exercise Px

Exercise Px is a clinical exercise prescription project built around the 7 Specifier Framework. It helps clinicians, students, and researchers think more clearly about exercise dose, intensity, structure, physiological targets, and behavioral health integration.

  • Clinical exercise prescription framework
  • METs, heart rate, breathing, timing, and structure
  • Designed for behavioral health documentation and education
