# Motus Salus Website

A simple static website for Motus Salus.

## Files

- `index.html` — the page structure and content
- `styles.css` — all visual styling
- `script.js` — small JavaScript file that updates the footer year

## Beginner GitHub Pages Setup

1. Create a new GitHub repository.
2. Upload these files to the repository root.
3. Go to Settings > Pages.
4. Under "Build and deployment," choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save.

GitHub will publish the site at a URL like:

`https://your-username.github.io/your-repo-name/`

## Editing

The main text is in `index.html`.
The colors are near the top of `styles.css` inside the `:root` block.
