# Motus Salus Site

A clean, static multi-page website for Motus Salus.

## Files

- `index.html` — homepage
- `exercise-px.html` — Exercise Px page with radar image
- `rural-culture.html` — rural culture page
- `contact.html` — contact/collaboration page
- `styles.css` — site styling
- `script.js` — footer year script
- `assets/specifier-radar-example.png` — Exercise Px radar image
- `.nojekyll` — tells GitHub Pages to serve the site directly

## GitHub Pages

Upload all files and folders to the root of a public GitHub repository. Then go to:

Settings → Pages → Deploy from a branch → main → /root

Includes about.html
